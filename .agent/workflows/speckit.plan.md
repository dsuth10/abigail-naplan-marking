# /speckit.plan

Use this command to create a technical implementation plan for a feature defined in `specifications.md`.

## Workflow

1.  **Analyze Specs**: Review `specifications.md` and any related research.
2.  **Socratic Review**: Ask clarifies questions if anything is ambiguous.
3.  **Generate Plan**: Create `plan.md` in the feature directory.
4.  **Validate**: Ensure the plan adheres to the constitutional principles in `.specify/rules`.

## Template
The plan should follow the format:
- Goal Description
- Technical Stack
- Proposed Changes (Files/Components)
- Verification Plan
