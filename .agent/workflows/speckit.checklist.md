# /speckit.checklist

Use this command to generate a project-specific quality checklist.

## Workflow

1.  **Context Check**: Review the current task and progress.
2.  **Generate List**: Create a checklist of items to verify (e.g., unit tests, SEO check, accessibility check).
3.  **Integrate**: Usually appended to `tasks.md` or a separate `checklist.md`.
