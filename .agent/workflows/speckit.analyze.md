# /speckit.analyze

Use this command to perform a project-wide audit against the constitution.

## Workflow

1.  **Audit Mode**: Check all code and documentation against `.specify/rules/`.
2.  **Identify Violations**: List any deviations from principles or tech stack.
3.  **Propose Fixes**: Provide actionable steps to align the project with its rules.
